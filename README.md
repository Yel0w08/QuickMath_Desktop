# QuickMath Desktop 

offline Windows application for training mental math.

## Features
-  Math operations (addition, more coming soon)
- XP and coin system
- In-app shop — unlock difficulties and other things
- Statistics tracking
- Local save system

## Status
> 🚧 Currently in active development

## Tech
- C# .NET 10
- Windows Forms
- SQL Server LocalDB (local-only, mono-user)

## Getting Started
Download the latest release and run `QuickMath-installer.exe`

QuickMath runs locally on the PC with a local SQL Server LocalDB database stored in the current Windows user profile. No remote server is required.
